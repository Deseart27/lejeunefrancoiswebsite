PDF exporter


PDF exporter est un tool répondant à une problématique de l'application Agent 360 de la société Yanport.
Cette dernière permet de générer un dossier d'étude pour un bien immobilier donné via une application web mais ne permet pas nativement l'export en format PDF.
Puisqu'il s'agit d'une situation frustante pour l'utilisateur et créant de la sollicitation Sales & Support, j'ai développé un tool permettant à un utilisateur d'exporter l'étude en format PDF en deux clicks.

Quelques détails

Une machine virtuelle

Le problème principal venait de la pluralité de configurations de nos utilisateurs : OS, navigateurs, versions... L'export donnait des résultats différents pour chaque configuration.
La solution : exporter le PDF via une machine virtuelle dont on maîtrise à 100% l'environnement.

Selenium

Grâce à son WebDriver "ChromeDriver", Selenium nous a permis de simuler le comportement d'un utilisateur imprimant le dossier d'étude vers une "sortie pdf" avec des paramètres précis.

Heroku

La machine virtuelle a été créée et déployée via Heroku, une solution de facilitation du déploiement de webs apps en Python. L'utilisateur devait simplement rentrer un lien dans un formulaire pour récupérer son étude en format PDF.

Création d'une API

Afin de faciliter la reprise de mon projet par l'équipe technique de Yanport, j'ai déployé une API via le framework REST de Django.
