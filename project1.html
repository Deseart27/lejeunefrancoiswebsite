Terms and conditions


Projet réalisé dans le cadre de la validation du bootcamp Data Science de Le wagon 🚃

Terms and conditions est une extension Chrome qui détecte intelligement les conditions générales d'un site et les synthétise en quelques lignes grâce à un modèle de Machine learning dédié.

Quelques détails techniques

Django

Grâce à son framework REST, Django nous a permis de créer une API flexible appelant un modèle de ML et un scrapper intelligent

Machine Learning

Les conditions générales récupérées sont nettoyées et résumées via Sumy, une librarie basée sur NLTK.

Déploiement via Heroku et Gcloud

Pour des raisons de performance et de maintanabilité, nous avons scindé le déploiement en deux hosts : Heroku (scrapper) et Gcloud(modèle de ML).

Selenium

Grâce à son WebDriver "ChromeDriver", Selenium nous a permis de créer un scrapper intelligent, lequel détecte les conditions générales d'un site et les extrait automatiquement.

Mon rôle au sein du projet

Architecture du back-end

Réalisation de l'architecture du back-end, à savoir organiser la communication entre notre extension (JS), le scrapper intelligent, notre base de données (GoogleBigQuery) et notre modèle de Machine Learning.

Création des APIs

Création de deux APIs dédiées via le framework Django REST. Chaque API possède des endpoints précis & dédiés (détection des conditions générales, extraction, vérification dans GBQ, nettoyage du texte, appel du modèle de ML...)

Déploiement de la solution

Mise en production du scrapper intelligent sur Heroku via Django for Heroku + gunicorn, et mise en production du modèle de ML via Gcloud Environnement flexible App Engine

Préparation de la présentation

Organisation et design des slides de présentation du projet via Canva.
